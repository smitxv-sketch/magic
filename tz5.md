**Задача для Front-end:**

* Внедрить искусственные задержки (`setTimeout` или хуки с задержкой).
* Процесс должен занимать реалистичные 1.5 – 3 секунды.
* **Для Агента-Секретаря:** Сначала должен пробегать сканер по тексту, затем пауза 0.5 сек, затем поля заполняются одно за другим.
* **Для Агента-Юриста:** Использовать эффект печатающегося текста (Typewriter effect) для вывода риск-репортов. Текст должен появляться побуквенно/построчно, как в ChatGPT. Это создает мощный гипнотический эффект для зрителей.

### 14. Техническое задание (Патч): TV-Scaling (Адаптация под большие экраны)

**Контекст:** Презентация почти наверняка будет выводиться на большой телевизор или проектор. Tailwind CSS по умолчанию оптимизирован под ноутбуки и десктопы. На 65-дюймовой панели текст может оказаться слишком мелким, а элементы — сбиться в кучу по центру.

**Задача для Front-end:**

* Проверить работу Плеера на разрешениях `2xl` (1536px и выше) и 4K.
* Использовать классы вроде `2xl:text-2xl` для базового текста и `2xl:max-w-7xl` для контейнеров, чтобы интерфейс масштабировался, а не просто висел маленьким прямоугольником в центре огромного экрана.
* Убедиться, что контраст изумрудного и янтарного цветов хорошо считывается издалека (избегать слишком тонких шрифтов — использовать `font-medium` или `font-semibold` для ключевых метрик).
