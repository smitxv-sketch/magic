# AI-Оркестратор СЭД

Демонстрационный стенд платформы интеллектуальной оркестрации для 1С:Документооборот.

## О проекте

Платформа представляет собой "умный слой" над классической СЭД, позволяющий встраивать AI-контролёров (LLM) в цепочки согласования документов.

### Основные возможности

1.  **Режим "Студия"**: Визуальный конструктор для настройки AI-кубиков (промпты, регламенты, правила).
2.  **Режим "Плеер"**: Интерактивная демонстрация прохождения документа по маршруту с реальным вызовом LLM.
3.  **Библиотека действий**: Каталог готовых реакций (возврат, эскалация, автосогласование).

## Технологический стек

-   **Frontend**: React 18, Vite, TypeScript
-   **State**: Zustand
-   **UI**: Tailwind CSS, Framer Motion, Radix UI
-   **AI**: Google Gemini Flash API (Client-side)
-   **Docs**: mammoth.js (docx parsing)

## Запуск

1.  Клонируйте репозиторий.
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите локальный сервер:
    ```bash
    npm run dev
    ```
4.  Откройте приложение в браузере.
5.  Введите ваш Google Gemini API Key в настройках (шестеренка в правом верхнем углу).

## Сценарии

В демо включены два сценария:
1.  **Согласование договора**: Проверка орфографии, юридических рисков и финансовых лимитов.
2.  **Исходящее письмо**: Проверка реквизитов и тональности ответа регулятору.
